# 📝 MERN Stack Todo Application

> A full-stack Todo List application built to efficiently track tasks and manage daily productivity. 

This project demonstrates a complete CRUD (Create, Read, Update, Delete) workflow using the MERN stack. It features a RESTful API backend communicating with a responsive React frontend.

---

## ✨ Features

*   **Create Tasks:** Easily add new todos to your list.
*   **Read Tasks:** View all active and completed tasks in real-time.
*   **Update Tasks:** Edit existing tasks or mark them as complete/incomplete.
*   **Delete Tasks:** Remove unwanted tasks from the database permanently.
*   **Responsive Design:** Clean UI that works seamlessly across desktop and mobile devices.

---

## 🛠️ Tech Stack

| Environment | Technology | Description |
| :--- | :--- | :--- |
| **Frontend** | React.js | UI rendering and state management |
| **Backend** | Node.js & Express.js | Server-side logic and REST API creation |
| **Database** | MongoDB & Mongoose | NoSQL database and object data modeling |
